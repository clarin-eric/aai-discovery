Exposed APIs:
.../rest/metadata/discojuice
Transform the shibboleth metadata files (configured in web.xml) into discojuice compatible json.
Parameters
None
.../rest/lookup/ip/{ip}
Perform a lookup in the geo ip database to retrieve the geo information for the supplied ip.
Parameters
  • {ip}, IP address to look up in the geo ip database
.../rest/lookup/url/{url}
Resolve the url to an ip and perform a lookup in the geo ip database to retrieve the geo information for the supplied ip.
Parameters
  • {url}, URL to resolve to an ip address and to look up in the geo ip database
